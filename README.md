# KataKita-Adventure
KataKita Adventure adalah game edukasi berbasis web yang dirancang untuk membantu pembelajaran Bahasa Indonesia melalui soal-soal interaktif. Game ini dilengkapi dengan fitur timer, skor, umpan balik benar dan salah, serta ranking pemain sehingga membuat proses belajar menjadi lebih menarik, menantang, dan memotivasi siswa untuk aktif belajar.
